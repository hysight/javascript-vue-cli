
# 开发

## 安装依赖
```
    npm install 
```

## 修改配置: config
```
    const development = {
        host: 'localhost',
        port: 2019
    };
    const production = {
        host: 'localhost',
        port: 8020
    };
```

## 启动命令

```
    // 运行一次即可
    npm run dll

    // 每次启动需要运行
    npm start
```
